# 🚀 Local AI Social Media Agent

An AI-powered workflow built with **n8n**, **Docker**, and **Ollama** that automatically generates complete social media content from a single topic.

Instead of manually researching, writing, and creating captions, the workflow orchestrates multiple AI-powered steps to produce ready-to-use Instagram/Reels content.

---

## Features

✅ Topic Research

✅ AI Research Agent

✅ Instagram Reel Script Generation

✅ AI Image Prompt Generation

✅ Instagram Caption Generator

✅ Hashtag Generation

✅ Fully Local LLM (Ollama)

✅ Built using n8n Automation

## Requirements

- Docker
- n8n
- Ollama

Install model

```
ollama pull phi3:mini
```

Run n8n

```
docker compose up
```

Import the workflow into n8n.
